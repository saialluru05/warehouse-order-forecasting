## Data Source

Download all raw files from Kaggle:
https://www.kaggle.com/competitions/store-sales-time-series-forecasting

Files needed:
- train.csv
- test.csv
- stores.csv
- holidays_events.csv
- oil.csv
- sample_submission.csv

Place all files in this folder before running the notebooks.
Run notebook 00 first if you want dates shifted to recent timeline.
